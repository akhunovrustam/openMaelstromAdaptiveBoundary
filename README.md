It is a clone of __openMaelstrom__ (opensource Multiplatform Adaptable Expressive and Lean Sph Toolkit for Realism Oriented Methods). The original repo can be find under https://github.com/wi-re/openMaelstrom. 

All installation instructions can be found in original repo.

New features implemented only in this repo
---
- boundary handling now includes also particle-based boundary
- decoupling is implemented for fluid-boundary interactions 
- adaptiveness of boundary particles is introduced for adaptive SPH
---
